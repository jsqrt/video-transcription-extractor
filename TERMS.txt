## 5. Privacy and Network Use

Describely is designed to run **offline**:

* No telemetry is sent.
* No usage analytics are collected.
* Network connections are blocked at the socket layer, except for
  loopback (`127.0.0.0/8`, `::1`) — used only to talk to a local
  Ollama instance if you have one running.
* The bundled Whisper model is included in the installer and never
  downloaded at runtime.

If you uninstall the Software, no data leaves your machine because none
ever did.

Transcripts and summaries produced by the Software are generated by
